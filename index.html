I am a PhD student of Programming Languages and Software Engineering at School of Computer Science, University of Technology Sydney, supervised by Dr Yulei Sui. My research interest includes Program Analysis, Software Security and application of Machine Learning in the above fields. I am now working with the open-source pointer analysis and program depedence analysis tool SVF and the open-source CFL-reachability analysis tool POCR.
